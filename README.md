**#🔍 PickSmart**

A responsive **product discovery website** designed to help users explore, search, and filter products efficiently through a clean and intuitive interface.

---

## ✨ What it does

PickSmart focuses on **discoverability, not transactions**.

Users can:

* Browse products
* Filter by categories
* Search products in real-time
* Navigate a responsive and user-friendly interface

---

## ⚡ Features

* 🔎 Live Search (instant filtering)
* 🗂️ Category-based filtering
* 🎨 Clean and modern UI
* 📱 Fully responsive design

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling & responsiveness
* **JavaScript** – Interactivity & dynamic filtering
